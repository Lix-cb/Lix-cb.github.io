1. HTML

Introducción

¿Qué es HTML?

Herramienta que se utiliza para la estructura de página web, es como el "esqueleto de una página". Define las seccciones y el texto que llevará cada componente.

Explica las tags de head y body

La etiqueta "Head" contiene información que el usuario no ve directamente (bueno, quizás sí ve el título de la pestaña), pero también incluye metadatos y la conexión con el CSS.

Mientras que el "body" contiene todo lo que el usuario ve, como textos, imágenes, botones, listas, etc.

Basic Tags

Explain the section, article, aside, header, footer tag

"section" se utiliza para dividir el contenido en bloques relacionados entre sí, es para seccionar.

"article" representa el contenido que podría no estar seccionado directamente.

"aside" es utilizado para contenido secundario.

"header" es el encabezado o parte superior de una página.

"footer" es el pie de página, la parte de hasta abajo de la página web.

What is CSS?

What is the purpose of CSS?

What is a selector? What is a CSS rule?

Indica qué elemento quiero modificar. Una reglla CSS es la de: selector + propiedades. En donde se indica qué quiero modificar y posteriormente las propiedades que se ven afectadas.

Basic CSS

Explain a:

Tag Selector

Selecciona el nombre de una etiqueta. Por ejemplo: h3{}

Class Selector

Selecciona elementos de una clase específica, ejemplo: .ParteDos{}

Tag and Class Selector

Selecciona una etiqueta con una clase específica, ejemplo: p.ParteDos{}

Id Selector

Selecciona un elemento único previamente definido con Id:"". Ejemplo: #carnitas{}

What is a class? What is an id? How are they related to CSS?

Una clase es un "molde" que pueden seguir ciertas secciones de una página, compartirán atributos de CSS.

Un ID debe ser único. Ambos se utilizan para aplicar estilos específicos con CSS.

Explain the CSS Box Model? Show an example.

Todo en CSS es una caja, un espacio que si no se indica no se visualiza directamente delimitado pero siempre está. Dentro de ella está el contenido.

Explain the media queries

Permiten cambiar el diseño segúin el tamaño de la pantalla. Se utilizan para páginas responsivas

Layout CSS

Explain display: grid

Grid se utiliza para crear estructuras en filas y columnas. El la propiedad que define un elemento como contenedor de una cuadrícula.

Explain display: flex

Flex se utiliza para alinear elementos. Permite distribuirlos de forma más flexible, por defecto los elementos se alinean horizontalmente.
